Mobile Health Support Application

Overview

The **Mobile Health Support Application** is an Android-based system designed to improve community access to essential healthcare information and basic services. It focuses on accessibility, simplicity, and usability, making it suitable for users with varying levels of digital literacy.

This application addresses common healthcare challenges such as:

* Limited access to reliable health information
* Difficulty in scheduling consultations
* Lack of awareness of available healthcare resources

By providing a centralized mobile solution, the app promotes proactive health management and improves communication between users and healthcare services.

Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Pawlo7777777/duhis.git
   ```

2. Open the project in **Android Studio**

3. Connect your Firebase project:

   * Add your `google-services.json` file to the `/app` directory

4. Build and run the app on an emulator or physical device

---

Firebase Configuration

Make sure the following services are enabled:

* Authentication
* Realtime Database

Update your database rules and indexes as needed for queries.
